# Two Tools for the Cyberpunk RED Table

Playing a lot of Cyberpunk RED lately. Built these two to smooth the table experience, especially when solo or running quick one-shots.

## Dice Roller and Initiative Tracker

Quick numbered buttons for standard rolls, but also a text input for custom expressions (2d6+1d8+3, whatever). CPR-specific mode where a natural 10 explodes into another d10 and a 1 fumbles. Full roll history stored so you can scroll back if someone argues about what was actually rolled. Initiative tracker rolls and sorts combatants, tracks turns and rounds, highlights whose turn it is. No server, just a clean interface.

## Screamsheet Generator

Generates a procedural Night City tabloid front page from random tables. Masthead, lead story, side stories, rumors, a corp ad promising chrome or synthehol. Toggle the tone between serious, absurdist, and sensationalist. Print button gives a clean layout for physical table use. It's essentially a random encounter generator disguised as a newspaper, useful for color, rumors to uncover, or just rolling to see what's in the news that session.

Both run offline, export results to clipboard so you can paste into your session notes, and print cleanly for physical tables. No subscriptions, no ads, just tools meant for running faster.
